> kirita怎么插入竖排文字

**Krita（很可能您指的“kirita”就是这个免费开源绘画软件）** 在最新版本（5.2 及以上）中支持竖排文字，但目前文字工具的界面尚未完全更新高级功能，需要通过 **SVG 源代码编辑器** 来手动设置。

### 操作步骤：
1. 选择 **文字工具**（快捷键 T），在画布上点击并拖动创建一个文字框（矩形区域）。
2. 松开鼠标后，会弹出文字编辑对话框，在这里输入你的文字内容（可以先输入横排文字）。
3. 关闭编辑对话框后，选中这个文字形状（矢量层）。
4. 在工具选项面板（通常在右侧）中，切换到 **“SVG 源代码”** 页面（或叫 SVG Source）。
5. 在 `<text>` 或 `<tspan>` 标签的 style 属性中，添加或修改为：
   ```
   writing-mode: vertical-rl;
   ```
   - **vertical-rl**：从右到左、从上到下竖排（适合日式漫画、中文古籍等传统竖排）。
   - 如果需要从左到右竖排，可以用 `vertical-lr`。
6. 为了实现自动换行，还可以添加：
   ```
   inline-size: 高度值px;  // 例如 inline-size: 500px; 根据你的文字框高度调整
   ```
7. 保存修改后，画布上的文字就会变成竖排显示。

### 注意事项：
- 这属于高级用法，目前 Krita 的文字工具还在迭代中（未来版本如 5.3 或更高会直接在界面上添加竖排按钮）。
- 如果需要更复杂的竖排（如漫画对白），有些用户会安装第三方插件，比如 GitHub 上的 “krita-verticalsvg-convert-plugin”（搜索即可找到，适合日文竖排转换）。
- 另一种临时 workaround：单独输入每个字符，然后手动旋转 90 度并排列，但这很麻烦，不推荐多文字时使用。

如果您用的不是 Krita，或者有具体版本号/截图，可以再补充，我帮您进一步确认！

> 是这个软件，那怎么弄那种漫画里常见的圆圆的字体，以及黑芯白边的样式

在 Krita 中制作漫画常见的**圆圆的（圆润粗体）字体** + **黑芯白边（黑色填充 + 白色描边）** 样式，主要靠以下步骤实现。Krita 的文字工具支持这些效果，非常适合漫画对白。

### 1. **选择圆润的漫画字体**
   - 漫画里常见的“圆圆的”字体通常是粗体（Bold）、圆角（Rounded）的 sans-serif 或专用漫画字体。
   - 推荐免费/常用字体（直接在系统安装后，Krita 就能用）：
     - **Anime Ace** 或 **CC Wild Words**：经典美式漫画字体，粗圆、易读。
     - **Komika** 系列（Komika Display、Komika Text）：圆润大胆，适合日漫风格。
     - **Action Man** 或 **Billy**：圆角粗体，活泼感强。
     - 日漫常用：**IPAmj明朝** 或 **Anime-style fonts** 如 “Wild Words” 变体，但中文竖排时推荐 **DFP丸ゴシック** 或 **Noto Sans Rounded**（Google Fonts 下载）。
     - 下载地址：Dafont.com、Google Fonts 或 Blambot.com（漫画专用字体，很多免费商用）。
   - 安装后重启 Krita，在文字工具选项里就能选到。

### 2. **创建文字并设置黑芯白边**
   - 用文字工具（T）在画布上拖出一个文字框，输入内容。
   - 选中这个矢量文字层（必须是矢量层）。
   - **添加白色描边（Stroke）**：
     1. 右键文字层 → **Layer Styles**（图层样式）。
     2. 勾选 **Stroke**（描边）。
     3. 设置：
        - **Size**：3~8 px（根据字体大小调整，白边粗细，通常 4-6 px 最像漫画）。
        - **Position**：Outside（外侧描边，避免覆盖黑色填充）。
        - **Color**：白色（#FFFFFF）。
        - **Fill Type**：Solid Color。
     4. 文字本身颜色设为黑色（在文字编辑器或工具选项里改）。
   - 结果：文字中间黑色填充（芯），外面一圈白色边，看起来醒目突出，超级像日漫/美漫对白。

   - 如果想更粗的白边：把文字层放进一个组（Ctrl+G），然后给组也加一个更大的白色 Stroke，就能叠加多层边（黑芯 + 内白 + 外白）。

### 3. **结合漫画气泡（Speech Bubble）**
   - Krita 自带气泡库：**Settings → Dockers → Vector Libraries**（矢量库），里面有 Comic Speech Bubbles，直接拖到画布上。
   - 推荐下载 Manga 风格气泡：
     - 搜索 “Manga Style Speech Bubble for Krita” （Gumroad 上有免费的 kaichi1342 套装），下载 SVG 文件。
     - 放进 Krita 的 Symbol 文件夹（Settings → Manage Resources → Open Resource Folder → symbol），重启后就在矢量库里。
   - 把文字层放到气泡下面，或用剪切组（Clipping Group）让文字限制在气泡内。
